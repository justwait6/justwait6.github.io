4.踩踩小坑


1.Servlet.service() for servlet [jsp] threw exception java.lang.NullPointerException

原因分析: 如果判断传入的爱好(favorites)是否等于null的话,出现这样的异常,
因为我们在后边调用了foreach对favorites进行遍历,而没有考虑它为null的可能性

解决办法: 判断favorites是否为空,若为空,则添加一个”未填写”字符串到里面,最后输出
一个更好的办法是,若为空,则不调用foreach字符串


2. According to TLD, tag [jsp:getProperty] must be empty, but is not
原因分析: 估计只有粗心的小伙伴会犯介样的错

解决办法: 加一个因**原因漏掉的斜线 “/“

更换为

3.Servlet.service() for servlet [RegisterServlet] in context with path [/UserLogin] threw exception

java.lang.IllegalStateException: Cannot forward after response has been committed

原因分析: 从一个页面跳到另一个页面, 然而却报出”响应被提交之后不能forward”,原来,我之前为了测试,在前边写了一个

解决办法: 把这些打印到页面的输出注释掉
