Workbench导入sql文件

发表于 2018-04-20

Workbench导入.sql文件


工具环境:

材料: 一个sql的script

0.问题描述


使用Workbench建立了一个数据库, 名叫shopping

现在有一个script,script里是一张创建好的表以及该表包含的一些表项
要把这张表导入到shopping当中

1.解决思路


1.如果对mysql命令熟悉的话,直接对着cmd敲命令行就搞定了,但是俺记不住,还得找半天资料

2.使用辅助工具,如本文里的Workbench.这也是我的选择

2.遇到困难


创建一个新的数据库shopping还挺简单
工具图标栏 –> create a new schema in the connected server(三个圆柱叠起来那个) –> 名称,字符集

然后是运行.sql脚本(资源来自慕课网教学视频).如下

直接file –> Run SQL script –> (设置好之后)Run, 汇报如下错误:

Preparing…
Error executing “mysql.exe” –defaults-extra-file=c:\users\vector\appdata\local\temp\tmprmnfsh.cnf –protocol=tcp –port=3306 –host=localhost –user=root:
[Error 267]

原因: 没有去折腾,hah

解决办法:

file –> open SQL script –> 编辑脚本, 在脚本开始处添加use shopping(use 数据库名) –> 点击文本框左上角的小闪电图标

3.小结


Workbench比命令行好用

好玩,哈哈
